More information

So what happened? Well, when you access a webpage, your browser contacts the server using the HTTP protocol.
The server, in response, returns some code, and the browser tries to display it.
In the simplest case, this "code" can be just words - the browser will display them as such.
However, to create page structure (and cool effects!) we use languages like HTML.
When a browser receives a response from the server that looks like HTML code, it tries to build a page from the HTML code.
This allows for pages with scripts inside them, which allows for interactive pages. Which, today, is the vast majority of pages.
